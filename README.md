# CyberShift: The Rogue Data

Minimalist neon rogue-lite puzzle runner built with Flutter & Flame Engine.

## Project Vision
A fast-paced offline mobile experience where the player controls a rogue data packet escaping a collapsing system.

## Tech Stack
- **Framework:** [Flutter](https://flutter.dev)
- **Engine:** [Flame](https://flame-engine.org)
- **State Management:** Flame Components
- **Persistence:** [Hive](https://pub.dev/packages/hive) (for offline rogue-lite upgrades)
